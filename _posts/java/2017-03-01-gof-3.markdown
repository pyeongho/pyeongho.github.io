#### 7.2 추상 팩토리 패턴
  - GUI 팩토리가 좋은 예
  - 각각의 운영체제에 맞게 동작 시키기
  - 팩토리인스턴스를 이용해서 원하는 객체를 가지고 오는 흐름
  - 맥,윈도우,리눅스의 환경에서 해당 팩토리들이 공개 되면 좋지 않다.
    - 맥에서 개발중인데 리눅스 팩토리 사용하고 하는 현상 발생
  - 각각의 기능을 팩토리인스턴스로 이동시킨다.
  - 라이브러리 형태로 제공 이라고 생각 해야 한다.
